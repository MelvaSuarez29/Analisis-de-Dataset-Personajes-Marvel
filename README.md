# Analisis-de-Dataset-Personajes-Marvel
## Análisis Exploratorio de Datos — Personajes Marvel Comics
### Actividad: se realiza el análisis exploratorio de datos (EDA) sobre personajes del universo Marvel utilizando Python.
### Fuentes de datos
- TXT delimitado por `;`
- Excel (`.xlsx`)
- JSON (API REST simulada)
- MongoDB (exportación JSON)
- SQLite
- SQL Server (`.sql`)

### Contenido del análisis
- Integración y concatenación de datos desde 6 fuentes
- Creación de columna de trazabilidad `source`
- Análisis exploratorio inicial
- Limpieza y tratamiento de datos
- Detección de valores atípicos mediante IQR
- Análisis por décadas, alineación, sexo y estado vital
- Visualizaciones estadísticas y comparativas

### Tecnologías utilizadas
- Python 3.9+
- Pandas
- Matplotlib
- Seaborn
- SQLite3
- JSON
